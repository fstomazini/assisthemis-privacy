# Política de Privacidade – Assisthemis GPT

Esta Política de Privacidade descreve como os dados são tratados ao utilizar o
Assisthemis GPT, um assistente jurídico personalizado integrado ao sistema Assisthemis.

## 1. Dados tratados
Ao utilizar o Assisthemis GPT, poderão ser tratados os seguintes dados, conforme
informado voluntariamente pelo usuário:

- Número de processo judicial
- Texto de petições e manifestações jurídicas
- Informações fornecidas livremente na conversa

O GPT não coleta dados automaticamente nem acessa informações externas sem
interação explícita do usuário.

## 2. Finalidade do uso dos dados
Os dados são utilizados exclusivamente para:

- Geração de peças jurídicas
- Exportação e salvamento das petições no sistema Assisthemis
- Organização e vinculação das petições aos respectivos processos

## 3. Compartilhamento de dados
Quando o usuário solicita explicitamente o salvamento de uma petição, os dados
informados são enviados ao sistema Assisthemis.

Nenhum dado é compartilhado com terceiros além do serviço OpenAI e do próprio
sistema Assisthemis.

## 4. Armazenamento e segurança
Os dados enviados ao Assisthemis seguem as políticas internas do sistema e as boas
práticas de segurança aplicáveis.

## 5. Direitos do usuário
O usuário pode solicitar esclarecimentos, correções ou exclusão de dados conforme
a legislação aplicável (LGPD).
